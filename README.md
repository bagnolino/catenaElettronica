# Progetto Catena Elettronica

Questo repository contiene il materiale relativo a un progetto di laboratorio di elettronica.

## Descrizione

Il progetto consiste nell'analisi di una semplice catena elettronica, composta da un circuito formatore (shaper) e uno stadio di amplificazione. L'obiettivo è studiare il comportamento dei singoli componenti e della catena completa, sia nel dominio del tempo che della frequenza.

## Contenuto del Repository

- **`AnalisiBagno.ipynb`**: Un Jupyter Notebook che contiene tutta l'analisi dei dati sperimentali, i fit e i grafici realizzati in Python.
- **`presentazione.pdf`**: La presentazione finale del progetto che riassume i risultati ottenuti.
- **`datiExcel/`**: Questa cartella contiene i dati raccolti durante l'esperimento e salvati in formato `.csv`.
- **`LtSpice/`**: Contiene i file di simulazione dei circuiti realizzati con LTspice.
- **`Parte2/`**: Contiene dati grezzi salvati direttamente dagli strumenti di laboratorio (es. oscilloscopio).
